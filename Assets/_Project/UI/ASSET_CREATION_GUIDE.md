## üñºÔ∏è Asset #1: Wooden Board Background

### **Specifications:**
- **File Name**: `OwareBoardBackground.png`
- **Size**: 1920x1080 pixels (or 2048x1024)
- **Format**: PNG (no transparency needed)
- **Style**: Light brown wooden texture

### **What to Create:**
A rectangular wooden board texture with:
- Light brown color (#D1B28C or similar)
- Optional: Wood grain texture
- Optional: Subtle shadows at edges
- Clean, flat design (like a table surface)

### **Quick Methods:**

**Option A: Photoshop/GIMP** (5 minutes)
1. Create new document: 1920x1080
2. Fill with light brown (#D1B28C)
3. Add noise: Filter > Noise > Add Noise (5-10%)
4. Add subtle gradient overlay (darker at edges)
5. Optional: Apply wood texture overlay
6. Save as PNG

**Option B: Free Online Tool** (2 minutes)
1. Go to: https://www.canva.com (free)
2. Create 1920x1080 design
3. Use brown rectangle
4. Add "Wood" texture from elements
5. Download as PNG

**Option C: Use Free Texture** (1 minute)
1. Download from: https://www.textures.com (search "light wood")
2. Or: https://unsplash.com (search "wood texture")
3. Crop to 1920x1080
4. Adjust brightness to be light brown

**Option D: Solid Color** (10 seconds - START HERE!)
1. Create 1920x1080 image
2. Fill with #D1B28C (light brown)
3. Done! (You can replace later with textured version)

### **Where to Save:**
```
Assets/_Project/UI/Sprites/OwareBoardBackground.png
```

## üï≥Ô∏è Asset #2: Pit/Hole Sprite

### **Specifications:**
- **File Name**: `PitHole.png`
- **Size**: 256x256 pixels
- **Format**: PNG with transparency
- **Style**: Circular dark hole with depth

### **What to Create:**
A circular pit that looks like a hole carved into wood:
- Dark brown center (#3D2817)
- Gradient from dark center to slightly lighter edges
- Optional: Subtle inner shadow for depth
- Round shape (perfect circle)
- Transparent background

### **Quick Methods:**

**Option A: Photoshop/GIMP** (3 minutes)
1. Create 256x256 document
2. Select circular marquee (perfect circle)
3. Fill with dark brown (#3D2817)
4. Apply inner glow: dark brown, soft
5. Add radial gradient: darker at center
6. Delete background layer (make transparent)
7. Save as PNG

**Option B: Figma** (2 minutes)
1. Create 256x256 frame
2. Draw perfect circle (hold Shift)
3. Fill with radial gradient:
   - Center: #2A1710
   - Edge: #3D2817
4. Add inner shadow effect
5. Export as PNG

**Option C: Simple Circle** (30 seconds - START HERE!)
1. Create 256x256 image
2. Draw filled circle
3. Fill with dark brown (#3D2817)
4. Apply radial gradient (darker center)
5. Save with transparent background

### **Where to Save:**
```
Assets/_Project/UI/Sprites/PitHole.png
```

## üîµ Asset #3: Marble Sprites (3 Colors)

### **Specifications:**
- **File Names**: 
  - `MarbleRed.png`
  - `MarbleBeige.png`
  - `MarbleBrown.png`
- **Size**: 128x128 pixels each
- **Format**: PNG with transparency
- **Style**: Glossy sphere/ball

### **What to Create:**
Three shiny marble sprites that look like glass balls:
- Sphere shape (perfect circle)
- Glossy/shiny appearance
- Highlight on top (light reflection)
- Slight shadow at bottom
- Three colors:
  1. **Red/Terracotta**: #E37359
  2. **Beige/Tan**: #F3E0C4
  3. **Dark Brown**: #432717

### **Quick Methods:**

**Option A: Photoshop/GIMP** (5 minutes per marble)
1. Create 128x128 document
2. Draw perfect circle (hold Shift)
3. Fill with base color
4. Apply layer styles:
   - Bevel & Emboss (smooth, depth 50%)
   - Inner Glow (white, soft)
   - Gradient Overlay (lighter at top)
5. Add white ellipse at top-left (highlight)
6. Add dark shadow at bottom
7. Make background transparent
8. Repeat for 3 colors
9. Save each as PNG

**Option B: Figma** (3 minutes per marble)
1. Create 128x128 frame
2. Draw perfect circle
3. Fill with base color
4. Add radial gradient (light at top)
5. Add white circle at top-left (20% size, 50% opacity)
6. Add blur effect to highlight
7. Export as PNG
8. Repeat for 3 colors

**Option C: Simple Gradient Circles** (1 minute - START HERE!)
1. Create 128x128 image
2. Draw filled circle
3. Apply radial gradient:
   - Top: Lighter version of color
   - Bottom: Darker version of color
4. Add small white circle at top-left (highlight)
5. Save with transparent background
6. Repeat for 3 colors

**Option D: Download Free Assets** (30 seconds)
1. Search "glossy marble sprite" on:
   - https://opengameart.org
   - https://itch.io/game-assets/free
2. Download and recolor to match our colors
3. Resize to 128x128

### **Colors to Use:**

**Marble 1 - Red/Terracotta:**
- Base: #E37359
- Lighter: #FF9980
- Darker: #C85A40

**Marble 2 - Beige/Tan:**
- Base: #F3E0C4
- Lighter: #FFFFFF
- Darker: #D9C4A0

**Marble 3 - Dark Brown:**
- Base: #432717
- Lighter: #66402E
- Darker: #2A1710

### **Where to Save:**
```
Assets/_Project/UI/Sprites/MarbleRed.png
Assets/_Project/UI/Sprites/MarbleBeige.png

```

## ‚öôÔ∏è Import Settings in Unity

After creating assets:

1. **Select each sprite in Unity**
2. **In Inspector:**
   - Texture Type: **Sprite (2D and UI)**
   - Sprite Mode: **Single**
   - Pixels Per Unit: **100**
   - Filter Mode: **Bilinear**
   - Compression: **None** (for quality) or **Normal Quality**
3. **Click Apply**
